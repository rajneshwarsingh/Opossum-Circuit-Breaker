# Opossum Circuit Breaker

## Overview

This repository demonstrates how to implement the Circuit Breaker Pattern in Node.js using Opossum to improve microservice resilience, fault tolerance, and service reliability.

## Why Opossum?

Opossum is a powerful Node.js library that: 
- Prevents cascading failures
- Helps handle timeouts / latency issues
- Automatically switches between open / closed / half-open states
- Improves stability in distributed systems / microservices
